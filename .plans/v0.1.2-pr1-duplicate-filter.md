# v0.1.2 PR-1 — 3-class DUPLICATE filter

Tiny bug fix. Keep this plan short. No CEO ceremony, no Codex audit, no traceability matrix. If this plan grows past 150 lines, you're overthinking it.

**Scope anchor:** `.plans/TODOS.md` v0.1.2-T1. **Ground truth:** `reference/NEMWEB-STRUCTURE.md §2.1.1` (12 `_LEGACY` placeholders / 23 multi-file stragglers / 617 GBB files).

## Problem

`scripts/extract_patterns.py` mirror-walk treats every `/DUPLICATE/` listing identically. v0.1.1's filter shape causes 17 market-report datasets to emit a `path_template` that points into `/DUPLICATE/` instead of the real parent. PR #9 flipped this to an unconditional skip, which regressed 640 real files (617 GBB + 23 stragglers). Both shapes are wrong.

## Fix

The `_LEGACY` suffix is the discriminator. Skip the listing only when every file in the parsed listing matches the `_LEGACY.zip` placeholder class *and* the parent path includes `/DUPLICATE/`.

Three classes per §2.1.1:

| Class | Shape | Action | path_template |
|---|---|---|---|
| (a) | listing under `/DUPLICATE/`, every file ends in `_LEGACY.zip` (12 dataset dirs) | **skip listing** | parent path (real data lives one level up) |
| (b) | listing under `/DUPLICATE/`, mixed non-`_LEGACY` files (5 subtrees, 23 files) | **keep** | actual path incl. `/DUPLICATE/` |
| (c) | `GBB/DUPLICATE/` rolling archive (617 files) | **keep** | actual path incl. `/DUPLICATE/` |

## Code change

`scripts/extract_patterns.py` — insert guard in the `main()` mirror-walk loop after `files = parse_listing(idx, parent_path)` (current line 480) and before the `if not files:` branch.

```python
# §2.1.1 three-class DUPLICATE model: skip only class (a) — listings under
# /DUPLICATE/ where every file is a _LEGACY.zip placeholder. Class (b) multi-file
# stragglers and class (c) GBB/DUPLICATE/ rolling archive stay indexed.
if (
    "/DUPLICATE/" in parent_path
    and files
    and all(f["name"].endswith("_LEGACY.zip") for f in files)
):
    duplicate_legacy_skipped += len(files)
    continue
```

Declare `duplicate_legacy_skipped = 0` next to the other counters (line ~471). After the walk, log:

